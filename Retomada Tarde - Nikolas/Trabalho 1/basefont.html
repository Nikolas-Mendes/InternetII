Tag Descontinuada <basefont>:

A tag <basefont> ela serve para definir uma cor e tamanho para uma letra padrão, só não podemos usar este comando no HTML5. Como por exemplo: Forma que a tag <basefont> fica quando usada: T "Não deu certo pois este comando não funciona em HTML5".
Forma que a tag <basefont> é usada: <basefont color="red" size="5"> (letra) </basefont>
Hoje em dia essa tag foi substituída pelo arquivo CSS.
